# Lithium Market Intelligence Dashboard

An interactive Streamlit dashboard modeling lithium cost curves, supply-demand balance, and price formation across global assets.

## Key Features
- Asset-level cost curve (LCE basis)
- Supply-demand balance modeling
- Price formation based on marginal cost + scarcity
- Scenario analysis (EV adoption, LFP share)
- Identification of marginal producers

## Tech Stack
- Python (Pandas, NumPy)
- Plotly (visualization)
- Streamlit (UI)
- Modular modeling (cost, supply, demand, price)

## How to Run
```bash
pip install -r requirements.txt
streamlit run app/app.py
